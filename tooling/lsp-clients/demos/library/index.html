Library Demo

ES module import — for apps that bundle Monaco with vite, webpack, etc. Consumes @kson/monaco-editor as a third-party package.

1. Create an editor

createKsonEditor starts a language server in a Web Worker and returns the editor handle. 

import { createKsonEditor }
    from '@kson/monaco-editor';

const editor = await createKsonEditor(
    document.getElementById('editor'),
    {
        value: '{ name: "hello" }',
        lspOptions: {
            bundledSchemas: [{
                fileExtension: 'kson',
                schemaContent: mySchema,
            }],
            enableBundledSchemas: true,
        },
    },
);

2. Read and write content

The return value exposes the Monaco editor instance directly via editor.editor: 

editor.editor.getValue();
editor.editor.setValue('...');

3. Listen for changes

editor.editor.onDidChangeModelContent(
    () => { /* ... */ },
);

Change events appear in the log below as you type.

4. Clean up

editor.dispose();
// output appears here
